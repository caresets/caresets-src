Le contenu du Glossaire BeSafeShare est en cours de développement, et cette page est en construction.
{: .info-box .should}

Le Glossaire BeSafeShare est une collection de définitions approuvées utilisées dans les CareSets belges : ce sont des définitions génériques qui s'appliquent à chaque utilisation d'un terme donné. Par exemple, "BusinessIdentifier" est défini comme un identifiant métier (non technique) d'un CareSet, qui peut être un Patient, une AllergyIntolerance, une Vaccination, une MedicationLine, etc.
L'objectif d'un glossaire est de garantir que les informations entre les CareSets sont sémantiquement interopérables, c'est-à-dire que la signification des éléments de données reste alignée pour éviter les problèmes d'interopérabilité.

Le Glossaire présenté ici est un rendu de la source authentique, fournie par le RIZIV-INAMI dans le cadre du projet CareSets (voir la page [gouvernance](../governance) pour plus de détails).
Techniquement, il est implémenté sous forme de CodeSystem FHIR et est disponible dans le package Terminology.
